1 Curvel {#curvel}
2 ============
3 Curvel computes roadway elevations, grade, and crown slope with respect to a given vertical curve and superelevation transition. Alignment information can be transferred between Curvel, PGSuper and PGSplice.
4 
5 # Introduction
6 A Curvel project is started as described in [Creating BEToolbox Projects](@ref creating_projects). A wizard will guide you through the project definition process.
7 The stationing for the project is carried on a survey line called the "L" Line. The profile grade is carried on the Profile Grade Line (PGL). The PGL does not have to be coincident with the "L" Line. ![Roadway Section](roadway_section.png)
8 
9 <h2 id="Step1">Step 1 - Define Vertical Curve</h2>
10 The project definition begins with a vertical curve.
11 ![](step1.png)
12 To define the vertical curve
13 1. Enter the entry grade, g1, and the exit grade, g2.
14 2. Enter the curve length
15 3. Enter the station and elevation of the Point of Vertical Intersection (PVI)
16 4. Press the [Next] button to continue
17 
18 <h2 id="Step2">Step 2 - Define Roadway Section</h2>
19 Superelevation transitions can be included in the project definition.
20 ![](step2.png)
21 1. Check the "Correct for superelevation or crown" box
22 2. Enter the offset from the crown point to the profile grade line
23 3. Enter the superelevation data
24 4. Press the [Next] button to continue
25 
26 <h2 id="Step3">Step 3 - Define Results Locations</h2>
27 You specify the locations where elevations are computed. Elevations can be computed at individual station, at locations generated between two stations, and along skewed lines measured from the alignment.
28 
29 The input for all three location definition methods is done with simple grids. Use the [+] and [-] buttons to add and remove rows from the grids.
30 ![](step3.png)
31 
32 ### Individual Stations
33 Enter a series of individual stations where profile grade line elevations are to be computed.
34 
35 ### Station Range
36 Enter a series of starting and ending stations, along with the number of spaces between these stations, where profile grade elevations are to be computed.
37 
38 As an example, if you wanted to compute profile grade elevation every 100 feet between station 0+00 and 5+00, enter 0+00 as the start station, 5+00 as the end station, and 4 spaces between the start and end.
39 
40 ### Skew Line
41 Elevations offset from the profile grade line can be computed with Skew Lines. Horizontal curves can be included in the skew line definitions.
42 
43 To compute the elevation offset from the profile grade line:
44 1. Enter the station where the skew line crosses the "L" Line
45 2. Select the datum that describes how the offset is measured. The offset can be measured along a line that is normal to the "L" Line and measured from the crown line or along a line that is skewed with respect to the "L" Line and measured from the "L" Line. Refer to the figures below for details.
46 3. Enter the offset
47 4. Enter the skew angle of the line
48 5. Enter the radius of the "L" Line (enter 0.0 if the "L" Line is straight)
49 6. Enter the offset from the crown point to the "L" Line
50 7. Press the [Finish] button

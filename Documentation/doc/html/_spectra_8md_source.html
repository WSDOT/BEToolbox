1 Spectra {#spectra}
2 =====================
3 Spectra computes a design response spectrum using the Three-Point method defined in the AASHTO Guide Specifications for LRFD Seismic Bridge Design, 2nd Edition, 2011.
4 
5 Spectral accelerations are based on the 2014 USGS Seismic Hazard Map with 7% probability of exceedance in 75 years.
6 
7 Input Parameters
8 ----------------
9 
10 Item | Description
11 -----|------------
12 Site Coordinates | Enter that location of the site with decimal values of latitude and longitude. Site coordinates are limited to 24.6&deg; - 50.0&deg; N and 65&deg; - 125&deg; W.
13 Site Soil Classification | Use the drop down list to select the site soil classification.
14 [Update] | Press to update the design response spectrum.
15 [Export] | Exports the design response spectrum in a format that can be used by CSiBridge.
16 
17 Importing Design Response Spectrum into CSiBridge
18 -------------------------------------------------
19 To import the design response spectra into CSiBridge:
20 1. Press [Export] to export the spectra. Select a file name and location.
21 2. Start CSi Bridge and create a new project or open an existing project.
22 3. Select **Loads > Functions Type > Response Spectrum > Expand arrow**
23 4. Use the *Choose Function Type to Add* drop down and select **From File**
24 5. Press [Add New Function...]
25 6. Press [Browse...] and select the file you created in Step 1.
26 
27 > NOTE: The exported response spectrum file does not contain header lines
28 
29 > NOTE: The exported response spectrum file is in Frequency vs Value format (the default setting for CSiBridge)

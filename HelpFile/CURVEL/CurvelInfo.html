Curvel Information

Bridge Elevations Program

The program "Curvel" is intended to provide the Engineer with a rapid means of obtaining roadway elevations with respect to a given vertical curve and superelevation transition.

Input

Vertical Curve Data:

a) G1: Grade leading into the curve
b) G2: Grade leading out of the vertical curve
c) Elevation of the PI
d) Curve length
e) Station of PIVC
Note: Only one vertical curve is used at a time.  For a straight grade, list the PI elevations for a convenient point on the grade and list the curve length as 0.
 

Yes/No

a) "Yes" if superelevation corrections are desired.
b) "No" if not

SuperElevation Data

Offset from crown to profile grade line (feet).

Stations

Station 1: In feet
Station 2: In feet
Station 3: In feet
 

Slope Left

a) Slope left for Station 1 (ft/ft all)
b) Slope left for Station 2
c) Slope left for Station 3
 

Slope Right

a) Slope right for Station 1
b) Slope right for Station 2
c) Slope right for Station 3
 

Individual Stations Data

a) Station
b) Offset (Only when superelevation and crown point data are given).
 

Increment Data

a) "991" - Cannot be modified - for internal program use.
b) Initial Station
c) End Station
d) No. of Spaces
e) Offset
 

Skew Line Data (optional entries)

Note: The following are optional entries and are only used if the "Yes/No" option (second entry line of form) is selected to "Yes".
a) Code
    1) 992 - all following offsets will be taken as being normal to and measured to the right of the crown line.
    2) 993 - all following offsets except crown line to 'L' line will be taken as being along the skew line measured to the right from the 'L' line.
b) Station
c) Offset
d) Skew angle, Deg. Min. Sec.
e) Radius to 'L' line.
f) Offset dis., crown line to 'L' line (ft).
 
  Notes:
These notes are referenced on the entry form with (a., b., c., etc) next to the appropriate blocks.
a) This note no longer used.
b) Distances to the right of crown point are (+), to left are (-).
c) Leave blank if superelevation correction is not used.
d) If offsets are radial measured from crown line use "992".
    If offsets are long skew measured from "L" line use "993".
e) Skew angle sign is clockwise positive (See instructions above).
    For negative skew use (-) in degree field only.
f) Radius is (+) if alignment curves right.
g) Radial distance.

Torsion Information

Torsion Program

This program calculates torsional moments of inertia for bridge cross sections. Sections are limited to close cell box girders and WSDOT standard prestressed girders.

Description:

The methods and formulas used are described in the CALTRANS Design Manual, Sections C-2 thru C-12 and several other texts.

Input is free format and uses "BDS" cross section descriptions for the box girder data. A BDS file may be created from TORSION box girder data.

Prestressed girder input utilizing WSDOT girder sections is not related to "BDS" in any way. The prestressed girder input cannot be used to generate a BDS input file.

GirComp Information

GirComp Program

GIRCOMP is a program that calculates the section properties of a composite girder with either a rolled or built-up steel section.   

Input:

For a rolled steel section, please use the "Rolled GirComp" version of this program.  For a built-up steel section, please use the "Built-Up GirComp" version of this program.  This information file applies to both versions of the program.
    For a rolled section, the numbers input are the area, depth, and moment of inertia of the section and any number of lines with a concrete slab width, thickness, G distance (between the section top and slab bottom) and three values of N (the modular ratio = (E)steel/(E)concrete).
    For a built-up section you may input any number of lines that include the built-up beam web depth and thickness, top flange width and thickness, bottom flange width and thickness, the concrete slab width, thickness, and G distance and three values of N.
    You may enter only one rolled beam and any number of slab properties or any number of lines of built-up beam and slab properties per program run.  You may not combine rolled and built-up sections in the same run.
    The program uses standard engineering formulas to calculate the section properties.  The input data is print out and also the built-up girder I, S bottom and top, Y bottom and top and Q of the bottom and top flange and the area.  The composite properties output are for the built-up girder, the I, S bottom and top, Y bottom and top and Q of the bottom flange, slab and slab + top flange.  The properties output are the same for the rolled beam except that only the slab is output.

Curvel

The Curvel program is intended to provide engineers with a rapid means of obtaining roadway elevations with respect to a given vertical curve and superelevation transition. Roadway elevation, grade, and crown slopes are computed.

Describing the Vertical Profile and Superelevation Transition

A simple three step wizard is used to define the vertical profile, superelevation transition, and locations where elevations are to be computed. The wizard is automatically started when a new Curvel project is created. You can start the wizard by pressing the Edit button in the results view.

 

Input Format

The following sections describe the input format for certain types of parameters. The general form for each input parameter is given along with examples. The general form uses a special notation as described below:

 

The items in a list of options are separated by a vertical bar, |. Choose one item from the list. For example, if L|R is given in the general form, you must enter either the L or the R.

 

Items that are inside parentheses are optional and need not be given. For example, if (+|-) is given in the general form you may enter either + or - or omit the symbol entirely.

 

The letter 'n' is used to indicate an ordinal number. The letters d, m, and s are used with angular measure and indicate degrees, minutes, and seconds, respectfully. Where spaces are used in the input format, use only one space.

Station

Stations can be input in on of the following formats:

 

Decimal Value

General form: (+|-)nnn.nn

Example: 234.34, -543.98

 

Stationing Value

General form: nn+nn.nn

Example: 2+34.34, -5+43.98

Offset

Offset distances can be input in one of the following formats:

 

Signed Value

General form: (+|-)nnn.nn

Examples: 12.0, -15.4

 

Directed Value

General form: nnn.nn L|R

Examples 12.0 R, 15.4 L

 

Remarks: Offsets to the right have positive values, offsets to the left have negative values.

Skew Angle

Skew angles can be input in one of the following formats:

 

Signed Decimal Value

General form: (+|-)nnn.nn

Examples: 13.455, -22.23

 

Signed DMS Value

General form: (+|-)dd mm ss.ss

Examples: 13 27 18, -22 13 48

 

Directed Decimal Value

General form: nnn.nn L|R

Examples: 13.455 R, 22.23 R

 

Directed DMS Value

General form: dd mm ss.ss L|R

Examples: 13 27 18 R, 22 13 48 L

 

Remarks: Skew angles to the left (counter-clockwise) have positive values, skew angles to the right (clockwise) have negative values.

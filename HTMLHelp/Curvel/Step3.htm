Step 3 - Define locations where elevations are computed

Elevations can be computed at individual stations, at equal spacing between two stations, and along skews.

 

Use the + button to add locations. The new location will be a copy of the last location defined in the grid.

Select one or more rows in the grid by clicking on the left hand column and use the - button to remove locations.

Individual Stations

Enter the location where individual elevations are to be computed. Enter the station and offset from the crown point to the desired point in the Individual Stations grid. The offset parameter is not used when corrections for superelevations are not considered.

Station Range

Enter the start and end station, number of spaces, and offset from the crown point to the desired point in the Station Range grid. The station range is divided into equally spaced locations and elevations are computed at the specified offset.

Skew Line

Enter the location where individual stations are to be computed along a line skewed from the normal to the L-Line. The offset can be measured from and normal to the crown line or it can be measured from the L-Line and along the skew.

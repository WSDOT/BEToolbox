Describing the Column

The column is described using the input fields at the top of the project window.

 

The input fields are described below

Section

Describe the column cross section.

Diameter

Diameter of the circular column

f'c

Concrete strength

 

Reinforcement

Cover

Reinforcement cover measured from the face of the column to the center of the reinforcement

As

Total area of reinforcement

Es

Modulus of elasticity of reinforcement

fy

Yield strength of reinforcement

 

Units

Select the system of Units (US or SI)

Update

The column interaction results will automatically update when the cursor moves from one field to another. Press the Update button to force an update at any time.

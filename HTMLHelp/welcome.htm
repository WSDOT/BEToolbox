Welcome to BEToolbox™

BEToolbox™ is a collection of bridge engineering utility programs. Many of these programs date back to the earliest days of computers at the WSDOT Bridge Office. Even though these programs are old and limited in their capabilities, they still prove to be useful and valuable tools for the practicing bridge engineer.

Getting Started

Getting started is easy, just create a new BEToolbox project. BEToolbox projects are created from the New Projects dialog.

 

  1. Start PGSuper (soon to be known as BridgeLink)

  2. Select File | New

  3. Select the BEToolbox project type

  4. Select one of the BEToolbox templates

  5. Press the OK button to create a new project

 

BEToolbox Programs

Geometrics

CURVEL - This program provides engineers with a rapid means of obtaining roadway elevations with respect to a given vertical curve and superelevation transition.

 

Section Properties

BoxGird - This program calculates section properties for box girders

GenComp - This program calculations section properties for a general composite section.

GirComp - This program calculates section properties for rolled beam and built-up composite sections.

Substructure

UltCol - This program computes ultimate moment capacity interaction diagram for circular concrete columns.

Compatibility with Previous Versions of BEToolbox

This version of BEToolbox is not compatible with previous versions. Input files created with previous versions cannot be opened by this version.

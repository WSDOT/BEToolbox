BoxGdr

BoxGdr is a program for computing the geometric section properties of a box girder.

 

Input

Use the input grid to enter the dimensions of the box girder.

 

Cross sections can be added by pressing the + button. Select a row by clicking on the row header in the left-most column and pressing the - button to delete a cross section.

 

Select US or SI units.

 

Press the Update button to update the output table.

Output

The output table lists all of the input dimensions of the box girder and the following section properties:

I

Moment of Inertia

YT

Location of neutral axis from top of girder section

YB

Location of neutral axis from bottom of girder section

AREA

Cross sectional area

WEIGHT

Weight per unit length based on an assumed unit weight of 160 lbs/ft3

ZT

Top section modulus

ZB

Bottom section modulus
